# Device Management POC

This is a Spring Boot based POC for device management.  
It includes basic CRUD APIs, Redis caching, Kafka event publishing, and JWT-based security.

## How to Run
1. Copy `application-example.yml` to `application.yml`
2. Add your MySQL, Redis, Kafka configs
3. Start MySQL, Redis, Kafka

## Technologies Used
- Spring Boot
- Spring Security (JWT)
- Redis Cache
- Kafka (Spring Cloud Stream)
- MySQL
